# 个人的毕业设计（仿猫眼的电影购票网站）
>  数据全部是从猫眼上爬取的。

## 简要介绍：
>  后台采用的Python的flask框架，前端采用的Vue框架。因为UI是模仿的猫眼，所以就没用UI框架。支付功能调用的是支付宝沙箱接口，登录是使用的手机验证码登录，不需要注册。

## 功能：分电影分类，搜索、预览、电影评论、电影购票、用户登录、后台管理......
